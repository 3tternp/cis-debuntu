# cis-debuntu

This is the bash shell script which is used to check CIS-benchmark test for Ubuntu & 
Debian based Linux distribution. 

# usage 
```
git clone https://github.com/3tternp/cis-debuntu
cd cis-debuntu/
chmod +x cisdebuntucheck.sh
./cisdebuntucheck.sh
```
